1. Для изменение цвета используется функция "color: ( цвет );". Сайт с цветами для html.

Далее функия для создания фона в блоке body - "background-color: rgba(0 - 255, 0 - 255, 0 - 255, 0 - 1)". Эта функция создает цвета и прозрачность цвета. Но так же можно изпользовать просто "background: (цвет)". Стоит заметить, что функция "color" в body меняет НЕ ФОН, а цвет текста в это блоке. На этом с цветами всеwef.



2. Для изменения фона (картинки), используется функция "background-image: url (ссылка на картинку ), url( можно несколько картинок );". Если картинка меньше чем нужно и повторяется, то используется функция "background-repeat: ( repeat, no-repeat, repeat-x, repeat-y );".

Функция "background-position: ( bottom, top, right, left ) или ( значение в px );" может принимать 2 значения, по вертикали и горизонтали. Если фонов несколько, то значения для них указываются через запятую и в порядке их добавления.

Функция "background-attachment: fixed;" зафиксирует положение фона внезависимости от изменения рамера и положения страницы. Если фонов несколько, то значения для них указываются через запятую и в порядке их добавления.

Чтобы задать рамер фона, используется функция "background-size: ( значение в px или % ) или ( contain - растянется во весь контейнер, cover - заполнит весь контейнер );". Если фонов несколько, то значения для них указываются через запятую и в порядке их добавления.

Для изменения цвета фона функция "background-color: (цвет);".

Можно указать все эти свойства разом, в определенном порядке, в функции "background: attachment color url position repeat, url( если нужно второе, то убираем цвет и ставим перед url запятую );" БЕЗ ЗАПЯТЫХ.



3. Для изменения отступа используется функция "margin: ( отступ в px );", "margin-( left, right, top, bottom );", для отступа с отпределенных сторон. Так же есть функция для изменения типа блока "display: ( inline, block, inline-block );". Функция "padding" для внутреннего отступа. Так же, вот ссылка на полезный видос, там про позиционирование и скроллинг ограниченного dir.



4. Для изменения цвета текста используется функция "color: ( цвет );".

Для изменения шрифта текста, используется функция "font-family: ( название шрифта );".

Чтобы измененить расстояние между буквами, используется функция "letter-spacing: ( размер в px );".

Для изменения размера текста, используется функция "font-size: ( размер в px );".

Чтобы подчеркнуть текст, используется функция "text-decoration: ( underline, overline, none );".

Для создаения тени у текста, используется функция "text-shadow: ( смещение по горизонтали ) px ( смещение по вертикали ) px ( размытие ) px ( цвет )", все значения указываются без запятых.

12. Для определения позиционирования, используется функция "position: ;" со следующими значениями "relative, absolute, fixed".

Absolute - абсолютное позиционирование относительно body.

Relative - позиционирование относительно родительского элемента.

Fixed - зафиксированное позиционирование.

16. Создание анимации начинается с установки ключевых кадров правила "@keyframes". Правило @keyframes содержит имя анимации элемента, которое связывает правило и блок объявления элемента. Ключевые кадры создаются с помощью ключевых слов from и to ( эквивалентны значениям 0% и 100% ) или с помощью процентных пунктов, которых можно задавать сколько угодно.

Для обращения к анимации нужно указать ее имя в параметре "animation-name: ( имя );", который указан в том элементе в CSS, который будет анимироваться.

Продолжительность анимации - "animation-duration: ;"..

Временная функция "animation-timing-function: ( ease, linear, ease-in, ease-out, ease-in-out );".

Повтор анимации "animation-iteration-count: ;". Для бесконечной анимации значение "infinite".

Направление анимации "animation-direction: ;". alternate - Анимация проигрывается с начала до конца, затем в обратном направлении. alternate-reverse - Анимация проигрывается с конца до начала, затем в обратном направлении. normal - Значение по умолчанию, анимация проигрывается в обычном направлении, с начала и до конца. reverse - Анимация проигрывается с конца.

Кратка запись "animation: animation-name animation-duration animation-timing-function animation-delay animation-iteration-count animation-direction;"

Множественные анимации "div { animation: shadow 1s ease-in-out 0.5s alternate, move 5s linear 2s; }".
